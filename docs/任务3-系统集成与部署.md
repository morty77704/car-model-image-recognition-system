# 任务3：系统集成与部署

## 系统架构

```
用户浏览器 (http://localhost:8080)
        │
        ▼
FastAPI 后端服务 (port 8080)
    │       │       │
    ▼       ▼       ▼
 EfficientNet  ResNet50  MobileNetV3
  (组员1)      (组员2)    (组员3)
        │       │       │
        └───────┼───────┘
                ▼
         共识投票 (Majority Vote)
                │
                ▼
         前端结果展示
```

## 技术栈

| 层级 | 技术 | 说明 |
|---|---|---|
| 后端框架 | FastAPI + Uvicorn | 高性能异步 Python Web 框架 |
| 深度学习 | PyTorch + torchvision | 模型加载与推理 |
| 前端 | 原生 HTML/CSS/JS | 无框架依赖，轻量高效 |
| 图片处理 | Pillow | 图片解码与预处理 |

## 部署方式

### 本地部署（Windows）

```bash
# 1. 安装依赖
cd backend
pip install -r requirements.txt

# 2. 启动服务
python -m uvicorn main:app --host 0.0.0.0 --port 8080

# 或双击启动.bat
```

### Docker 部署（可选）

```dockerfile
FROM python:3.11-slim
WORKDIR /app
COPY backend/requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt
COPY backend/ .
COPY frontend/ ../frontend/
EXPOSE 8080
CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8080"]
```

### 服务器部署

- 支持部署到云服务器（阿里云/腾讯云等）
- 使用 Nginx 反向代理 + Gunicorn/Uvicorn
- 模型权重文件约 127MB/个，需预留足够存储空间

## API 接口

| 方法 | 路径 | 功能 |
|---|---|---|
| GET | /api/health | 系统健康检查 + 模型状态 |
| GET | /api/models | 模型列表与详情 |
| POST | /api/recognize | 上传图片，三模型并行识别 |
| POST | /api/feedback | 用户反馈收集 |
