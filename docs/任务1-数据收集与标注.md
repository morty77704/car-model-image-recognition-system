# 任务1：车型图像数据收集与标注

## 数据来源

使用 **Stanford Cars Dataset**，这是学术界标准的车型识别基准数据集。

- **来源网站**: Stanford University AI Lab
- **数据集规模**: 8,144 张训练图像，8,041 张测试图像
- **类别数**: 196 个车型类别
- **覆盖品牌**: 49 个汽车品牌（涵盖德系、日系、美系、韩系、英系等）
- **标注方式**: 每张图片精确标注到品牌 + 车型 + 年款三个层级

## 数据预处理

1. **格式统一**: 所有图像统一转换为 RGB 格式
2. **尺寸调整**: 根据模型架构调整输入尺寸（300×300 / 224×224）
3. **数据增强** (训练时):
   - 随机裁剪 (RandomResizedCrop, scale 0.7-1.0)
   - 随机水平翻转 (RandomHorizontalFlip)
   - 色彩抖动 (ColorJitter: brightness/contrast/saturation ±0.2)
4. **归一化**: ImageNet 标准 (mean=[0.485,0.456,0.406], std=[0.229,0.224,0.225])
5. **数据集划分**: 85/15 训练验证随机切分

## 标注格式

```
品牌 车型描述 年份
例如: Audi TTS Coupe 2012
      BMW X5 SUV 2007
      Tesla Model S Sedan 2012
```

## 补充数据渠道

为丰富数据多样性，还可以从以下渠道收集：

- 汽车之家、懂车帝等国内汽车垂直网站的用户实拍图
- 4S 店展厅实拍
- 车展现场拍摄
- 街拍抓取

补充数据需手动标注为品牌+车型+年款格式。
